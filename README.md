# Adidas Landing Page

A modern and responsive landing page built with **HTML5** and **CSS3**. This project recreates a clean Adidas-inspired homepage while practicing core CSS concepts such as Flexbox, pseudo-elements, media queries, and responsive design.

## 📸 Preview


## 🚀 Features

* Responsive layout for desktop and mobile devices
* Built with pure HTML and CSS
* Modern glassmorphism-inspired container
* Flexbox layout
* Decorative background using `::after` and `clip-path`
* Navigation bar with hover effects
* Clean typography and spacing

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* CSS Pseudo-elements (`::after`)
* CSS `clip-path`

## 📚 What I Practiced

During this project, I learned and practiced:

* Creating page layouts with **Flexbox**
* Resetting default browser styles
* Styling navigation bars
* Using `margin` and `padding`
* Working with `width` and `height`
* Creating responsive designs with Media Queries
* Using `position: absolute`
* Layering elements with `z-index`
* Creating custom shapes using `clip-path`
* Adding hover animations with `transition`
* Applying transparent backgrounds using `rgba()`

---

## 📱 Responsive Design

The page automatically adjusts its layout for smaller screens.

* Desktop: Text and image are displayed side by side.
* Mobile: Content stacks vertically for a better user experience.
